# 🏋️ Workout Record App (v1)

간단한 운동 기록 웹 애플리케이션입니다.  
무게, 횟수, 세트를 입력하면 총 운동 볼륨을 계산하고 카드 형태로 기록합니다.

---

## 📌 Features

- ✅ 무게 × 횟수 × 세트 자동 계산
- ✅ 운동 기록 카드 생성
- ✅ 총 볼륨 표시
- ✅ 기본적인 DOM 조작 및 이벤트 처리 구현

---

## 🧮 Volume Formula

Total Volume = weight × reps × sets

예시:
66kg × 2회 × 3세트 = 396kg
